# Suggested commands

- `pnpm install` – install dependencies
- `pnpm build` – clean + build ESM + CJS + types
- `pnpm clean` – remove dist (uses rm -rf; on Windows use PowerShell or WSL)
- `pnpm lint` / `pnpm lint:fix` – eslint on src
- `pnpm prettier` / `pnpm prettier:fix` – check/fix formatting
- `pnpm test` – jest
- `pnpm typecheck:tests` – type-check tests
- `pnpm release:rc` – prerelease version bump
